Authentication Security: Two-Factor Authentication

Concept: Multi-Factor Identity Verification

CIA Principle Protected: Confidentiality & Integrity

Technical Brief

  • ▹ **Algorithm:** Combines a Knowledge Factor (Password) with an Ownership Factor (Mobile Device).
  • ▹ **Mechanism:** Utilizes Time-based One-Time Passwords (TOTP) valid for a short window (e.g., 30s).
  • ▹ **CIA Focus (Integrity):** Specifically defends account integrity against brute-force and credential stuffing attacks.

Step-by-Step Flow

  1. User submits username + password.
  2. **TechNova Cloud Server** validates first factor.
  3. Server pushes unique TOTP to user's registered device.
  4. User enters code; server performs multi-factor validation.
  5. **Code Verified ✓** confirms final authorization.
