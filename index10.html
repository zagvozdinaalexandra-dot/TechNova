Security Mapping:
The CIA Triad

Connecting Defenses to Core
Cybersecurity Principles

Confidentiality
Only authorised people can access the data
Integrity
Data cannot be secretly changed or corrupted
Availability
Systems are accessible when needed
Threat / Device Principle Protected Defense Rationale
Router — Hacker Confidentiality Firewall prevents unauthorized reading of private network data.
Dev PC — Virus Integrity Antivirus stops malware from modifying or corrupting source code.
Finance — Phishing Confidentiality Spam filter prevents credential theft and financial data exposure.
Manager — VPN Confidentiality Encryption prevents interception of remote data on public internet.
Cloud Server — 2FA Conf. & Integrity Stops unauthorized logins from viewing or corrupting company records.
Guest Wi-Fi — VLAN Confidentiality Ensures visitors have zero visibility into internal company systems.
